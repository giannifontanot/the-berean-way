Crear usuario nuevo

Elige una serie original y crea la carpeta del alumno. La herramienta copia el original tal cual (su contenido, header y footer) y solo cambia el nombre. Para algo especial, edita el data.js del alumno directamente en GitHub.

Datos del usuario

Si lo dejas vacío se usa «Los estudios de NOMBRE». Se copia desde sites/templates/<serie>/data.js.
