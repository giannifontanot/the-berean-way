Extractor de Google Sites

Copia el contenido de un Google Site a tu portapapeles en formato de texto estructurado, listo para convertirlo en lecciones de The Berean Way.

✅ Sí extrae el texto dentro de acordeones. Antes de copiar, el script abre automáticamente todas las secciones plegables y además lee el texto oculto como respaldo, así que no se pierde nada aunque esté colapsado.

Cómo usarlo

1
Arrastra el botón a tu barra de marcadores Mantén presionado el botón de abajo y arrástralo hasta la barra de favoritos de tu navegador (Chrome, Edge o Firefox). Solo necesitas instalarlo una vez.

Arrastra este botón a tu barra de favoritos:

📋 Extraer contenido

Si no ves la barra de favoritos: Ctrl+Shift+B (Windows) · Cmd+Shift+B (Mac)

2
Abre el Google Site en otra pestaña Ve a sites.google.com/view/los-estudios-de-jennifer/ y navega a la página (lección) que quieres copiar.
3
Haz clic en el marcador “Extraer contenido” El script abre todos los acordeones, espera un momento, extrae el texto (incluido el que estaba colapsado) y lo copia al portapapeles. Te muestra una vista previa para confirmar.
4
Pega el resultado aquí abajo Vuelve a esta página y pega el texto. Luego cópialo al chat con Claude para que lo convierta en lecciones del formato data.js.

Pegar el contenido extraído

Después de usar el marcador en cada lección, pega el resultado aquí. Puedes pegar varias lecciones seguidas.
